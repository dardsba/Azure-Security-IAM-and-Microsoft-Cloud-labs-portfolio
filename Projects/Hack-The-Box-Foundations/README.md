Overview

This project summarizes my foundational hands-on practice in Hack The Box, including completed Academy modules and Starting Point / Tier-based guided machines.

The purpose of this work was to build practical familiarity with Linux, networking, web applications, enumeration, basic exploitation concepts, and troubleshooting in lab-style environments.
Completed Academy Modules

I completed the following Hack The Box Academy modules:

    Linux Fundamentals
    Introduction to Bash Scripting
    Introduction to Networking
    Introduction to Web Applications

These modules helped strengthen my understanding of operating system basics, command-line usage, networking concepts, and web application fundamentals from a security perspective.
